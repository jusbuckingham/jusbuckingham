# Jus Buckingham | Entrepreneur • AI Engineer • Producer

Los Angeles, CA

Jus Buckingham builds modern software products at the intersection of technology, business, and culture. With roots in theatre and music — and a lifelong background as a saxophonist — he approaches engineering with the mindset of a creator: structured, intentional, and performance-oriented.

He is the founder of:

- **[LegatePro](https://legatepro.com/)** – A modern command center for estate and probate workflows, designed to bring clarity, structure, and intelligence to complex legal processes.
- **[Kofa.ai](https://kofa.ai/)** – A conscious news platform providing AI-curated reporting and thoughtful, independent analysis.

This repository contains the source code for his personal website, built to reflect that product-focused philosophy.

---

## 🛠 Tech Stack

- **Next.js** – React framework optimized for performance and SEO  
- **TypeScript** – Type-safe development for reliability and long-term scalability  
- **Tailwind CSS** – Utility-first styling for clean, responsive design  
- **Vercel** – Deployment platform with seamless CI/CD integration  

---

## ✨ Site Principles

- Minimal, refined interface  
- Accessibility-first structure  
- SEO-optimized metadata and structured data  
- Performance-focused builds  
- Clear product positioning (LegatePro + Kofa.ai)

## 🚀 Local Development

```bash
git clone https://github.com/jusbuckingham/jusbuckingham.git
cd jusbuckingham
npm install
npm run dev
```

Visit `http://localhost:3000`
